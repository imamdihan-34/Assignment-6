# 1.PROJECT NAME:
DigiTools Platform

# Description:
A modern, sleek, and highly interactive digital products marketplace designed to supercharge creator workflows. This platform allows users to browse premium digital assets and manage their cart seamlessly with a premium, Figma-inspired user interface.
# Technology I use:
This project is built using modern frontend technologies to ensure speed, responsiveness, and a great developer experience:

* **React.js** - Component-based UI architecture.
* **Tailwind CSS** - For rapid custom styling and smooth responsive layouts.
* **React Toastify** - To provide beautiful, non-intrusive user feedback.
* **JSON Data** - For dynamic data rendering without hardcoding elements.
* **Vite** - Next-generation, lightning-fast frontend tooling.
# Key Features:
* **Dynamic Product Toggling:** Users can instantly switch between the "Products" grid and their "Cart" view via a custom-designed animated pill toggle, preventing page reloads.
* **Real-Time Cart Management:** Interactive "Buy Now" triggers and a fully functional removal system with active count badges and cost totals.
* **Premium Aesthetic & Layout:** Features high-fidelity design elements including smooth boundary rounded corners (`rounded-[2.5rem]`), soft floating shadows, and vibrant gradient buttons.
